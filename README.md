

#  **Tableau Dashboard**

The dashboard provides:

###  **Readmission vs Clinical Variables**

* Diagnosis categories
* Age bucket
* Gender
* Race
* Medications
* Comorbidities

###  **Interactive Parameter**

User can dynamically select **ANY dimension** to analyze:

> *Example: “Show readmissions by Diagnosis Category”*
> *Example: “Show readmissions by Race”*

###  **Stacked bar charts**

Visualize distribution of:

* `<30`
* `>30`
* `NO`

### **Heatmap for Multiple Interaction**

* Diagnosis categories
* Age bucket
* Gender
* Race
* Medications
* Comorbidities
* Number of Records, Inpatient, Emergency

###  **Interactive Parameter**

User can dynamically select **ANY two dimension** together to analyze
