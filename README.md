#  **Dataset**

Data Source: [Diabetes 130-US Hospitals (1999–2008)](https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008)

This is a public Diabetes Dataset from UCI Machine Learning Repository. It has records from 130 US Hospitals from 1999 to 2008. 

#  **Tableau Dashboard**

The dashboard provides:

###  **Readmission vs Clinical Variables**

* Diagnosis categories
* Age bucket
* Gender
* Race
* Medications
* Comorbidities

###  **Interactive Parameter**

User can dynamically select **ANY dimension** to analyze:

> *Example: “Show readmissions by Diagnosis Category”*
> *Example: “Show readmissions by Race”*

###  **Stacked bar charts**

Visualize distribution of:

* `<30`
* `>30`
* `NO`

### **Heatmap for Multiple Interaction**

* Diagnosis categories
* Age bucket
* Gender
* Race
* Medications
* Comorbidities
* Number of Records, Inpatient, Emergency

###  **Interactive Parameter**

User can dynamically select **ANY two dimension** together to analyze
